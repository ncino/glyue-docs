# Value Mappings

A Value Mapping is essentially a lookup, where if a "From" value is encountered, it's replaced with its corresponding "To" value.

## Value Mapping Sets

A Value Mapping Set encompasses multiple Value Mappings and has various settings associated with it that change the behavior of it's mappings.

### Settings

When creating a new or editing an existing Value Mapping Set, there are various settings that can be configured.

<figure><img src="../.gitbook/assets/ETL Value Mapping Set Settings.png" alt=""><figcaption></figcaption></figure>

1. The name of the Value Mapping Set.
2. Checkbox that sets whether or not "From" values should be compared case sensitively.
3. Checkbox that enables a default value if no "From" value is matched.
4. The default value if no "From" value is matched.
